# .github

This repository contains GitHub-specific files for the https://github.com/wildfly-security['wildfly-security' GitHub organization]:

* https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file[Default community health files] for repositories in the organization.

Note that the .gitignore, dco.txt, LICENSE and SECURITY.md files in this repository are not default community health files; their presence here does not remove the need for equivalent files in other repositories in the organization.
